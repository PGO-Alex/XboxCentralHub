# XboxCentralHub

A Desktop APP for manage xbox rent in some minor business on mexico

This app is runs on .Net Framework 4.7, is builded with full C# and their use is focused on windows platform

to run this app you need to build it with Visual studio 19(preferently)

This app needs you have instaled Mysql or any LAMP, MAMP, XAMPP, AMPPS or WAMP instance

and running MySQL server

# Things working

- Add Console
  
- Delete Console
  
- Edit Console
  
- Display Console's
  
- Update Console list
  
- About App dialog

- Registry of new Rent Event's

- PDF file creation with all day rent entries

- Set custom app color & custom text app color

- Set on App Config windows a custom connection string or use the connection string builder
